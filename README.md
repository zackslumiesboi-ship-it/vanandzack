# vanandzack

Landing page for **Van + Zack the AI editor** — a weekly newsletter about AI tools that move SEO traffic.

## Stack

- Static HTML + CSS. No build step.
- Deployed via Cloudflare Pages → auto-deploys on every push to `main`.

## Edit + ship

```sh
git pull
# edit index.html / style.css
git add -A
git commit -m "your message"
git push
```

Cloudflare rebuilds in ~30 seconds.

## Files

- `index.html` — the page
- `style.css` — styles
- `README.md` — this file
